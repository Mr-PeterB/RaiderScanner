# Raider Scanner v2.0

## Installation:
```
https://github.com/Mr-PeterB/RaiderScanner.git
sudo ln -s $(pwd)/RaiderScanner/raider /usr/local/bin/
```
***
## Usage:
[+] Usage: ./raider [host file]

EXAMPLE:
./raider Desktop/hosts.txt
