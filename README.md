# Raider Scanner v2.0

## Installation:
```
git clone https://github.com/Mr-PeterB/RaiderScanner.git
sudo ln -s $(pwd)/RaiderScanner/raider /usr/local/bin/
```
***
## Usage:
raider [hosts file]

EXAMPLE:
raider Desktop/hosts.txt
